# NJIT Scheduling

Code and models related to **scheduling** (e.g. course scheduling, resource allocation, or job scheduling), developed in the context of NJIT (New Jersey Institute of Technology) or as standalone projects.

## What is in this repo

- Formulations (e.g. ILP, constraint programming) or heuristic algorithms for scheduling problems.
- Instances or data (if present) and scripts to run experiments.
- Documentation or reports (if present).

## How to run

1. Clone the repository and install dependencies (solver, Python/C++, etc.).
2. Run the main script or solver with the provided or your own instance.
3. See in-repo documentation for input format and parameters.
